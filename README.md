# Getting Started 
 - ### Clone this project
   - git clone git@github.com:Julio-Freitas/instaDog.git
  - ### Install of dependencies  
    - npm install or yarn 
  - ### In terminal project run
    - `npm start or yarn start`
- ### Ready, enjoy in
  - http://localhost:3000
